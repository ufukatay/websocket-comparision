# Websocket Comparision

Welcome to my websocket comparision program. 

To run Websocket1:

For server:
```
cd websocket1
python server.py
```
For client from another terminal:

```
cd Websocket1
python client.py
```

The stream should be started.

To run Websocket2:

For server:
```
cd websocket2
python server2.py
```
For client from another terminal:

```
cd Websocket2
python client2.py
```

The stream should be started.
